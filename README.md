# DetectaTela

App Android Kotlin para detectar TVs e telas externas (MediaRouter + DisplayManager).

Como usar:
1. Instale o APK
2. Abra o app e toque em "Buscar telas / TVs"
3. Conecte pela tela rápida do Android (Transmitir / Smart View)
4. Veja o status na tela e nos logs

Compilação via GitHub Actions (recomendado).
